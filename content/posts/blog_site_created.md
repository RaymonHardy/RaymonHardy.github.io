## Deployment:

### Hugo Local Install:
- Mac Terminal Installation and Start:
    - Brew Install Hugo
    - git init
    - git submodule add https://github.com/adityatelange/hugo-PaperMod/wiki/Features#profile-mode themes/hugo-PaperMod
    - hugo server -D
